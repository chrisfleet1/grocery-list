# Grocery List

An app for users to create their own shopping list, up to 24 items.

## Built With

- HTML
- CSS
- Javascript

## Instructions

1. Type in an item
2. Press 'Enter' key
3. The items are then added to a list
4. To clear a list, click the eraser icon

### Additional Notes

- Items can be crossed out, just by clicking on the items
